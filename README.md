# Emacs profiles

### GNU Emacs 26.2

first configured on Gentoo Linux.

## Usage

clone me as $HOME/.emacs.d

language server should be install manually for `lsp-mode`, including:

- gopls (go)

- clangd (C/C++)

- pyls (Python)

- rls (rust)

Or else disable hooks to start `lsp` on those major mode.
